# Caving In Kenya

Static Jekyll version of the former Weebly site for `cavinginkenya.com`.

## Local development

```sh
bundle install
bundle exec jekyll serve
```

The site is configured for GitHub Pages with `CNAME` set to `cavinginkenya.com`.

## Notes

- The original Weebly contact form was replaced with a direct `mailto:info@cavinginkenya.com` link because GitHub Pages cannot process form submissions without an external form service.
- Weebly runtime scripts, analytics snippets, customer-account code, and the Weebly footer were removed.
- CSS, uploaded images, and font files are served locally from this repository.
